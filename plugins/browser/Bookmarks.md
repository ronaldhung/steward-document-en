## Commands
### Find bookmarks
`trigger: bm`    
- `shift` supported
- `Batch operation` supported

#### Operating
- `Input` search
- `Enter` open the selected bookmark
    - `+ Shift` open the bookmarks with `shift` icon in bulk

### Delete bookmark
`trigger: bmd`

#### Operating
- `Input` search
- `Enter` delete the selected bookmark
